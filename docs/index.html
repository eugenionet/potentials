1.21.2.5 Movement in Vacuum

\(\eta = \frac{\tan\alpha}{\tan\beta} = \frac{c}{v_{Ph}} = 1\) \(\qquad\) \(2\pi = k \lambda\) \(\qquad\) \(v_{ph} = c\) \(\qquad\) \(\omega = k c\)

\(\varepsilon_0\mu_0 = \frac{1}{c^2}\)

1.21.2.6 Movement in Matter

\(\eta = \frac{\tan\alpha}{\tan\beta} = \frac{c}{v_{Ph}} = \sqrt{\frac{\varepsilon\mu}{\varepsilon_0\mu_0}} \neq 1\) \(\qquad\) \(v_{ph} = \pm \sqrt{\frac{1}{\varepsilon\mu} - \frac{\sigma^2}{4\varepsilon^2 k^2}}\) \(\qquad\) \(\omega = k v_{ph} = kc \sqrt{\frac{\varepsilon_0\mu_0}{\varepsilon\mu}} = kc\frac{\tan\beta}{\tan{\alpha}} = k \frac{c}{\eta}\)

\(\varepsilon\mu = \left( \frac{\eta}{c} \right)^2\)

Dispersion: \(k^2 = \varepsilon\mu\omega^2 + i \mu\sigma\omega\)

1.21.2.7 Near Field

\(r_0 \ll r \ll \lambda\) \(\qquad\) \(kr \ll 2\pi\) \(\qquad\) \(\frac{w_m}{w_e} \ll 1\)

\(r\) is small, fast movement

1.21.2.8 Far Field

\(r \gg \lambda\) \(\qquad\) \(kr \gg 1\) \(\qquad\) \(r \gg kr_0^2 \ll 1\) \(\qquad\) \(\frac{w_m}{w_e} = 1\)

\(r\) is big, slow movement
